# 使用结构体组织相关联的数据

<!-- https://github.com/rust-lang/book/blob/main/src/ch05-00-structs.md -->
<!-- commit 8a6130451b0817ead5c2522ce641dcb0f11a8571 -->

**结构体（struct）**是一个自定义数据类型，允许将多个相关的值封装在一起并为它们命名，从而形成有意义的组合。如果你熟悉面向对象的编程语言，你会发现结构体类似于对象的数据属性。在本章中，我们将比较元组和结构体的异同，并在此基础上进一步探讨，帮助你理解在何种情况下结构体是更好的数据组合方式。

我们还将展示如何定义和实例化结构体，并讨论如何定义与结构体类型相关的函数，尤其是被称为方法（method）的关联函数，这些方法指定了结构体应有的行为。通过结构体和枚举（enum）（将在第六章介绍），你可以在 Rust 程序中创建新的类型，并充分利用编译时的类型检查。
